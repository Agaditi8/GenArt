# GenArt

GenArt is a decentralized digital art marketplace built using Ethereum smart contracts.

## Tech Stack
- Solidity (Hardhat + Ignition)
- Ethers.js
- HTML, CSS, JavaScript
- IPFS (Pinata)
- Sepolia Testnet

## Features
- Connect wallet using MetaMask
- Mint and list digital art
- Buy listed art
- Fully on-chain ownership

## Network
- Ethereum Sepolia Testnet
